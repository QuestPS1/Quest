📱 Android Setup
  1. Download Virtual Hosts [Download]
  2. Install Virtual Hosts on your device
  3. Download hosts file QuestHost.txt
  4. Open Virtual Hosts → Select hosts file → Enable it
  5. Done! Server name: Quest
💻 Windows Setup
  1. Press Win + R, type C:\Windows\System32\drivers\etc
  2. Edit file hosts with Notepad
  3. Add these lines: 
    91.134.85.13 growtopia1.com
91.134.85.13 growtopia2.com
91.134.85.13 www.growtopia1.com
91.134.85.13 www.growtopia2.com
  4. Save file → Launch Growtopia
  5. Server name: Quest
🍏 iOS Setup
  1. Download Surge 5 from App Store
  2. Click Default.conf → Import Profile → Paste URL: iOS Host
  3. Allow VPN setup → Open Growtopia
  4. Server name: Quest
🍎 Mac Setup

Edit /etc/hosts file just like Windows and add the same IP addresses. Save → Launch Growtopia → Connect to Quest.
